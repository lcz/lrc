    使用文档

    基本：一行代码搞定：

    在使用时：用可以自定义的 css样式
    lrc_attention : 正在播放中的歌词选中样式
    lrc_moonlight : 基本样式

    以下必须的css 样式，可以直接复制到自己的项目中
    .lrc_attention{color:red;} .lrc_moonlight{color:#fff;} .lrc_hide > li{ display:none; } .lrc_show{ display:block; 

      
    lrc.build(参数一,参数二);
    
    
      
    参数一：
    
        ‘object’      页面内显示元素位置    id 属性
  	‘initTop’     歌词头初始位置
	‘center’      焦点位置
	‘empty’       构建头与尾部 的 空对象原始  
	‘isDropLrc’   启用歌词拖入，仅支持现代浏览器，  默认（true） 
	'lrcUrl'      歌词文件路径 ， 用 html5 Worker  对象读取的。
	'lrcText'     歌词文本    string

	'readType' ：  歌词解析方式  Read:弱一点，但是它快(默认)  ,vls1:功能类似悍马车一样，但开销贵一点
	’seekMark‘ ：  追踪方式   Value：慢一点 ， ValueInd：快一点

	’syntony‘   ： 歌词解析完毕后的 回调函数 function

	'media'：      播放器对象  默认支持  audio  video，   id 属性

	'open':        显示开启  默认：False
    
    
    
    
    
      
    参数二：
    
        'notoccupy'    // 普通样式	.class
	'occupy'      正在被播放中的歌词 加亮样式    .class
	'width' ：    宽度    如： 50px , 90%
	'height' ：   高度    如： 50px , 90%

	'align':      水平居中方式  默认：center
	'oneline'  ： 单行显示      默认：false

    
    


  如果你需要手动设置歌词到指定时间：
    
  lrc.setProgress(歌词对象, 时间);

  自定义歌词效果：
    
  lrc.open(歌词对象);  代开方法
  lrc.close(歌词对象);   关闭方法
  lrc.suspend(歌词对象);  暂停
  lrc.destroy(歌词对象);   歌词销毁 
  
  
  歌词对象的获取：（在调用build 方法时就会返回一个对象）
  	如：    var obj =  lrc.build({object:'lrc_1',media:'audio1'});
			obj: 就是对象啦！
  	
  
  歌词特殊标识：
    
	 sname :"ti",  //歌曲名
	 cname :"cl",  //作词
	 qname :"cs",  //作曲
	 bname :"ps",   //编曲
	 sgname :"ar",    //歌手名称
	 special :"al",   //所属专辑
	 kname :"by",		//歌词制作
	 end :"end"    //结束 语
	用户可以手动修改   如：lrc.tag.sname='title';
